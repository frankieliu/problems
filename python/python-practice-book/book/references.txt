Books and Tutorials
===================

* `Dive Into Python <http://diveintopython.org/>`_

Dive Into Python is a Python book for experienced programmers. It is available online.

* `Python Cookbook <http://www.oreilly.com/catalog/pythoncook/>`_

The Python Cookbook is a collection of problems, solutions, and practical examples for Python programmers, written by Python programmers.

Practice Problems
=================

* `Python Challenge <http://pythonchallenge.com>`_

    Python Challenge is a game in which each level can be solved by a bit of (Python) programming.

* `Project Euler <http://projecteuler.net>`_

    Project Euler is a series of challenging mathematical/computer programming problems that will require more than just mathematical insights to solve.
    Most of the problems can be solved by writing couple of lines of python code.

* `Ruby Quiz <http://rubyquiz.com>`_

    Ruby Quiz is a weekly programming challenge for Ruby programmers. Though it is designed for ruby, problems can be solved in any programming language.

    Some interesting problems from Ruby Quiz.
    
    * `Crosswords <http://www.rubyquiz.com/quiz10.html>`_
    * `Word Chains <http://www.rubyquiz.com/quiz44.html>`_
    * `Grid Folding <http://www.rubyquiz.com/quiz63.html>`_
    * `Happy Numbers <http://www.rubyquiz.com/quiz93.html>`_
    * `Housie <http://www.rubyquiz.com/quiz114.html>`_
    * `Morse Code <http://www.rubyquiz.com/quiz121.html>`_
